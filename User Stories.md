**Steward**  
**Must have**

* As a Steward, I want to mark issues as resolved, so that I can maintain an organized list and focus only on unresolved problems.  
* As a Steward, I want to receive notifications when an issue remains unresolved for an extended period, so that I can take timely action to ensure the issue is addressed.  
* As a Steward, I want to identify issues requiring attention in each park, so that I can monitor and manage the parks I supervise effectively.


**Should have**

* As a Steward, I want to be able to view and filter issues at the park that I supervise, so that I could manage my trails and track overall progress.  
* As a Steward, I want to retain all issues somewhere in the system, so that I can find a previous record.

**Could have**

* As a Steward, I want to be able to view a list of all volunteers, so that I could reach out to them.  
* As a Steward, I want to have advanced natural language processing by text, so that I would ask how many trees are down in north park.

**Trail User**  
**Must have**

* As a trail user, I want to access a trail reporting form, so that I could report issues on a trail.  
* As a trail user, I want to get a text once the issues that I reported gets resolved so that I could be more motivated to update the status of trails.  
* As a trail user, I want to sort issues based on which park they are at so that I could help out the park that I often go to.


**Could have**

* As a trail user, I want to be notified when issues I report are resolved, removed or deleted so I can stay up to date on my issue and overall community impact.

**Like to have**
